# Angulacia
-  Angulacia helps you to create a good structured folder architecture in your Angular project.

## Install

```console
$ npm i -g angulacia
```

Basic folder structure (angulacia -b)
--------------

```
└── src/
    └── app/
        ├── core/
        │   ├── guards
        │   ├── interceptors
        │   ├── models
        │   └── services
        ├── modules/
        │   └── index
        └── shared/
            ├── components
            └── libs
```
